Reason NOT to get a Windows PC

  1. Malware!: The biggest disadvantage of choosing a Windows over a Mac is that Windows is the most targeted operating system by malware because it is so widely used
  2. Lower cost = poor quality: Lower Costs laptops are often built with cheaper and often less reliable components, so this can impact the reliabiltiy and long-term performance
  3. Manufacturers: None of the hardware manufacturers have anything to do with the operating system, so each one must create specific drivers and figure out compatibility issues with the components that they use in each group
